Elegant Glassmorphism Elementor Addons including Card Grid and Advanced Button widgets.

== Description ==

Grid Glass Addons With Button is a lightweight Elementor addon plugin that provides modern glassmorphism UI widgets.

Features:
- Glass Card Grid widget with icon, title, description & button
- Advanced Glass Button widget
- Fully customizable from Elementor controls
- Clean, lightweight & performance friendly
- No external dependency required

Perfect for landing pages, SaaS websites, portfolios, and modern UI designs.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure Elementor is installed and activated
4. Use widgets from Elementor editor
